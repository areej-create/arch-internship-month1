Submitted by **Areej Sajjad** | UI/UX Designer & Frontend Developer

## Tasks

### Task 1 — Personal Portfolio
A fully responsive personal portfolio website built with pure HTML, CSS, 
and JavaScript. Includes sections for introduction, education, projects, 
skills, experience, testimonials, blog, achievements, and contact.

**Tech:** HTML · CSS · JavaScript

### Task 2 — To-Do List App
A full-stack to-do list application with a Flask REST API backend and 
SQLite database. Supports adding, editing, deleting, and filtering tasks 
by priority and status.

**Tech:** Python · Flask · SQLite · HTML · CSS · JavaScript

## How to Run Task 2
```bash
cd task2-todo-app/backend
pip install -r requirements.txt
python app.py
```
Then open `task2-todo-app/frontend/index.html` in your browser.
